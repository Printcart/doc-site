Platform Connect · WordPress

Connect WordPress / WooCommerce

Free plugin · install in ~2 minutes. Your credentials are on the right.

Connected · healthy
📦
Install method: Download the plugin ZIP, upload to WordPress, activate, then paste credentials.

Installation steps

  1. Download the Printcart plugin
    Compatible with WP 6.0+ and WooCommerce 7.0+.
  2. Upload to WordPress
    In WP admin: Plugins → Add New → Upload Plugin, choose the .zip, click Install Now.
  3. Activate the plugin
    After install, click Activate Plugin.
  4. Paste your credentials
    Go to PC Designer → General → Connect and paste Sid, Secret, and Unauth Token from the box on the right.
  5. Test the connection
    Click "Verify" on the WordPress side, or use the button below.

Platform Connect · Shopify

Connect Shopify

Public app (one-click) or Private app (custom credentials).

Option 1: Public app (recommended)

  1. Visit Shopify App Store
    Search for "Printcart Designer".
  2. Click "Add app"
    OAuth flow handles credentials automatically.
  3. Approve scopes
    Allow read/write access to products, orders, theme.
  4. Done
    Designer button auto-injects on product pages.

Option 2: Private app

  1. Open Shopify Admin
    Settings → Apps and sales channels → Develop apps.
  2. Create new app
    Name it "Printcart Designer". Set Admin API scopes.
  3. Install + get token
    Copy the Admin API access token.
  4. Paste here
    Use the credentials box on the right.

Connect Magento / Adobe Commerce

Composer install · supports v2 + v3.

composer require printcart/magento2-pod

Installation steps

  1. Install via Composer
    Run the Composer command above on your Magento server.
  2. Enable and configure module
    Run setup:upgrade and setup:di:compile in production mode.
  3. Open admin panel
    Go to Stores → Configuration → Printcart → API Settings.
  4. Paste credentials
    Enter Sid, Secret, and Unauth Token. Save config and flush cache.
  5. Verify connection
    The "Customize" button will appear on product pages.

Platform Connect · Custom API

Custom REST API integration

For developers building a headless or custom storefront. Embed the Printcart designer programmatically.

⌨
For developers. No plugin to install. Use REST API + Designer SDK to embed customization in any framework (React, Vue, Next.js, etc.).

Quickstart

  1. Install the Designer SDK 
    npm install @printcart/designer-sdk
  2. Initialize with credentials 
    import { PrintcartDesigner } from '@printcart/designer-sdk';

const designer = new PrintcartDesigner({
  unauthToken: 'YOUR_UNAUTH_TOKEN',
  containerId: '#designer-root',
});
designer.open({ productId: 'prod_123' });
  3. Server-side: list products / orders 
    curl https://api.printcart.com/v2/products \
  -H "X-Sid: YOUR_SID" \
  -H "X-Secret: YOUR_SECRET"
  4. Webhooks (recommended)
    Receive order events, design uploads, completion. Add a webhook URL in Apps & Connectors → Webhooks.
